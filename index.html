Free Code Camp - Timestamp Microservice

  1. I can pass a string as a parameter, and it will be checked to see weather or not that string is a valid unix timestamp or a natrual language date.
  2. If the parameter string is valid, a Unix timestamp and natural language form of the date will be returned.
  3. If the parameter string is invalid, null will be returned for both properties.

Example Parameters

Natural: https://jc-fcc-timestamp.herokuapp.com/December%2015,%202015

Unix: https://jc-fcc-timestamp.herokuapp.com/1450137600

Example Output

{ "unix": 1450137600, "natural": "December 15, 2015" }
